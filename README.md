# Desafio-java

## Ambiente local
1. Clone o projeto
2. Abra o mesmo na ide de sua preferência
3. Certifique-se que esteja utilizando o Java 11
4. Executar o comando mvn clean install para baixar as dependência relacionadas ao projeto
5. Para executar local, realizar configuração padrão do Java(Springboot) para executar a classe main( br.com.desafio.DesafioApplication).
6. Para acesso web http://localhost:8080/
7. É possível acessar o swagger retirando comentário no properties relacionado a ativação do "spring.profiles.active" e acessar a url 'http://localhost:8080/v1/api/swagger-ui.html'.
